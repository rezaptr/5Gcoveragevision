Help & Panduan Penggunaan

Platform web untuk perencanaan rute drive test dan simulasi kualitas jaringan 5G — membantu estimasi performa sinyal sebelum pengukuran lapangan dilakukan.

1 Alur Penggunaan Sistem

  1. Unggah file site dalam format XLSX yang berisi data koordinat dan konfigurasi antena.
  2. Pilih site (gNodeB) yang akan dianalisis dari daftar yang tersedia.
  3. Generate Ring sebagai batas radius perencanaan rute drive test.
  4. Generate Main Route kendaraan drive test menggunakan layanan routing OSRM.
  5. Jalankan simulasi RSRP, SINR, atau Coverage sepanjang rute yang terbentuk.
  6. Analisis dan evaluasi hasil simulasi pada peta interaktif.

2 Penjelasan Fitur Utama

Upload File Site (XLSX)

Digunakan untuk memuat data site jaringan yang berisi informasi koordinat, azimuth, dan beamwidth. Data ini ditampilkan pada peta sebagai penanda site dan sektor.

Generate Ring

Menghasilkan ring atau radius di sekitar site terpilih sebagai batas awal perencanaan rute drive test. Ukuran ring dapat disesuaikan.

Generate Main Route

Membentuk rute utama kendaraan drive test berdasarkan ring yang telah dibuat, menggunakan layanan routing OpenStreetMap (OSRM) secara realtime.

Simulasi RSRP / SINR / Coverage

Mensimulasikan estimasi kualitas sinyal jaringan 5G sepanjang rute drive test. Hasil simulasi divisualisasikan dengan peta berwarna berdasarkan kategori kualitas.

Komparasi & Evaluasi

Halaman Komparasi membandingkan hasil Drive Test asli dengan hasil simulasi. Halaman Evaluasi menampilkan distribusi error kualitas hasil simulasi per segmen jarak.

3 Informasi Status & Kendala

Status menunggu file site Menandakan sistem belum menerima data XLSX. Klik tombol Upload dan pilih file yang sesuai.

Route gagal dibuat Pastikan koneksi internet aktif. Sistem menggunakan layanan routing OSRM secara online untuk membentuk rute.

Site tidak ditemukan Pastikan nama site yang dicari sesuai dengan data pada file XLSX yang diunggah.

Data tidak valid saat upload CSV simulasi Pastikan file memiliki kolom: Lat, Lng, RSRP(dBm), SINR(dB), dan Distance(m).

4 Tips Penggunaan

  • Gunakan data koordinat site yang valid dan akurat agar hasil simulasi yang lebih akurat.
  • Pastikan koneksi internet stabil saat generate rute agar proses routing berjalan lancar.
  • Export hasil simulasi sebagai CSV agar dapat digunakan di halaman Komparasi dan Evaluasi.
